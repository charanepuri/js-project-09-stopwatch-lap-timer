# Stopwatch with Lap Timer ⏱️

## 📌 Project Overview

This project is a **Stopwatch Application** built using **HTML, CSS, and JavaScript**.
It allows users to measure time, pause/resume, reset the timer, and record lap times.

The stopwatch updates in real-time using JavaScript timing functions and displays time in **HH:MM:SS format**.

## ⚙️ How It Works

1. User clicks **Start**:
   - Timer begins using `setInterval()`

2. Time increases every second:
   - Seconds → Minutes → Hours

3. User can:
   - Pause the timer
   - Reset the timer
   - Record lap times

4. Lap times are displayed as a list.
5. Reset clears both timer and lap history.
